# Tokyo Band Simulator

A Roblox Studio rhythm game simulator set in Night Tokyo, where players manage their music band, buy clothing and instruments, hire musicians, and play rhythm minigames (XCNM keys).

## Structure
This project is configured with [Rojo](https://rojo.space/).

- **src/shared**: Shared module scripts (`SongData`, `GameSettings`).
- **src/server**: Server scripts and managers (`ServerController`, `DataManager`, `ShopManager`, `BandManager`, `ContractManager`).
- **src/client**: Client rhythm engine (`RhythmClient`).
